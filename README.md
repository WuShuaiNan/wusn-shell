# wusn-shell
- `达娃里氏`个人 Linux 管理面板。

## 项目简介
- 使用 shell 脚本命令封装的 Linux 管理面板，主要目的是为了提高 CentOS 7 环境下程序部署速度。

## 项目使用说明
- 手动上传 `wusn-shell` 文件夹到 `/usr/local/wusn-shell` 目录下。
- 运行 `chmod -R 755 /usr/local/wusn-shell`
- 运行 `./usr/local/wusn-shell/panel.sh`
- 愉快地使用吧~
